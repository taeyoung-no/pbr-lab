# PBR Lab

WebGPU renderer exploring physically-based rendering techniques.

---

## Dependencies

**System**
- Node.js 20+
- A browser with WebGPU support (Chrome 113+, Edge 113+)

**npm (auto-installed)**
- vite ^8.0
- typescript ~6.0
- @webgpu/types ^0.1
- wgpu-matrix ^3.4
- lil-gui ^0.21

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Output: `dist/`
